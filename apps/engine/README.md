# Running the backend

To install dependencies:

```sh
bun install
```

To run:

```sh
bun run dev
```

open [http://localhost:3000](http://localhost:3000) in your browser.

## Documentation

For API documentation, please refer to the [API Reference](http://localhost:3000/osmynt-api-engine/reference) page.
